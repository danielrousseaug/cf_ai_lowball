Lowball - Reverse Auction Platform
===================================

Project Overview:
-----------------
A platform where people bid DOWN for tasks. Lowest bidder wins and completes the work.

Core Features:
--------------
✓ Reverse auction bidding (lower = better)
✓ Multi-currency support (cash, points, favor tokens, time bank)
✓ Task creation and management
✓ User profiles with reputation scores
✓ Achievement system
✓ Leaderboards
✓ Real-time bid updates
✓ Buy It Now option
✓ Dutch auctions
✓ AI-powered chat assistant (Workers AI + Llama 3.3 70B)

Architecture:
-------------
Backend:
- Cloudflare Workers (edge compute)
- Durable Objects (persistent state)
- Workers AI (chat assistant)

Frontend (planned):
- Next.js 14
- React
- TanStack Query
- Tailwind CSS
- Framer Motion

Database:
- Durable Objects with SQLite
- In-memory Maps for fast access
